# TinyChat

一个小型聊天室

对于v0.1~v0.5, 参考 [https://socket-io-chat.now.sh/](https://socket-io-chat.now.sh/)

对于v0.6~?, 使用Semantic UI

## 更新日志

### v0.1

- 多房间聊天

- 需要注册

- 可以发送图片

- 可拓展命令工具

- 需要手动注册

- 无法删除用户、无法删除房间

- 后台可手动注册用户、注册房间
